# Restaurant Project

This is a simple PHP project I made while learning basic web development.

It includes:
- A restaurant ingredients page
- An "our mission" page
- Basic styling with CSS
- A local Apache setup using XAMPP

## Live Demo (Vercel)

https://resturentproject.vercel.app

## Project Pages

- Home: https://resturentproject.vercel.app
- Ingredients: https://resturentproject.vercel.app/ingredients.html
- Our Mission: https://resturentproject.vercel.app/our-mission.html

## Local Run (Apache + XAMPP)

1. Put the project inside the XAMPP htdocs folder.
2. Start Apache from XAMPP Control Panel.
3. Open in browser:
   - http://localhost/php/resturent_project/ingredients.html
   - http://localhost/php/resturent_project/our-mission.html

## Note

This Vercel deployment is serving the static pages (HTML/CSS/images).
If you add dynamic PHP server logic later, run it using Apache/PHP hosting.
